# Druv Prism

**AI-powered natural language analytics for Splunk and complex data sources.**

Druv Prism lets security and IT operations teams query their Splunk data 
in plain English — no SPL expertise required. Ask a question, get a 
verified answer, a chart, and the query that produced it.

## What it does

- **Natural language → SPL** — type a question, get a production-ready 
  Splunk query with no syntax knowledge needed
- **Automated schema discovery** — connects to your Splunk environment 
  and learns your indexes, sourcetypes, and fields automatically
- **Self-healing queries** — agentic retry loop fixes errors before 
  they reach you
- **Air-gapped deployment** — fully local LLM support via Ollama for 
  regulated industries (government, healthcare, financial services)
- **Continuous learning** — gets more accurate over time from your 
  team's query patterns

## Who it's for

SOC analysts, security engineers, IT Ops, and SRE teams who live in 
Splunk but spend too much time writing queries instead of investigating.
